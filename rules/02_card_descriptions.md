# II カードの説明文

カードに書かれている説明文に関する内容です。用語の意味や処理の順番で分からないときに参照してください。

---

## A カードの説明文について

- カードの説明文が基本ルールと違う場合は、**カードに書かれていることを優先**する。ただし変更後の効果が公表されている場合はその効果を適用。
- 「●●する」と「●●できない」が重なった場合、**「●●できない」を優先**。
- 従えない部分があっても使うことはできる。従える部分だけ行い、従えない部分は行わない。ただし「E-20 ●●する。その後、▲▲する」のように、一文を従えなかった場合はその後を行わないことがある。
- 数は任意で小さくすることは原則できない（D-04、D-05 に例外あり）。
- 対象の数が指定より少ない場合は、指定に一番近い数を適用する。

## B ダメージ計算関連

| 項目 | 内容 |
|------|------|
| **B-01** ●●ダメージ追加 | ワザのダメージをさらに大きくする。「+」付き。ダメージ計算の 1. で行う。 |
| **B-02** ●●の数×▲▲ダメージ | 対象の数とかけ算。「×」付き。1. で行う。結果が 0 なら 2. の効果があっても計算終了。 |
| **B-03** ●●ダメージぶん、このワザのダメージは小さくなる | 「-」付き。1. で行う。0 以下なら 2. があっても計算終了。 |
| **B-04** ダメージは「+●●」される | 2. ダメージを与えるポケモンが受けている効果で計算。複数ある場合は合計。 |
| **B-05** ダメージは「-●●」される | 5. ダメージを受けるポケモンが受けている効果で計算。複数ある場合は合計。 |
| **B-06** 弱点・抵抗力を計算しない | 3. 4. を行わずにダメージを与える。 |
| **B-07** かかっている効果を計算しない | 5. の効果と「ワザのダメージを受けない」を無視してダメージを与えられる。 |
| **B-08** 相手のポケモン●●匹に、(それぞれ)▲▲ダメージ | 指定数だけ選んでダメージ。ベンチは弱点・抵抗力を計算しない。 |
| **B-09** この（自分の）ポケモンにも、●●ダメージ | ワザを使ったポケモン自身もダメージ。通常どおり弱点・抵抗力・かかっている効果を計算。 |

## C 効果

| 項目 | 内容 |
|------|------|
| **C-01** トラッシュする | 指定のカードをトラッシュに置く。持ち主のトラッシュに置く。 |
| **C-02** ●●を（は）▲▲にもどす | 手札や山札に加える。持ち主の手札・山札に。場のポケモンをもどすとき、ダメカン・効果はなくなる。 |
| **C-03** 自分のバトルポケモンをベンチポケモンと入れ替える | 【にげる】と違いエネルギーをトラッシュしない。にげられない状態でも入れ替え可能。 |
| **C-04** 相手のバトルポケモンをベンチポケモンと入れ替える | 新たにバトル場に出すポケモンは**相手が選ぶ**。ワザの効果を受けないポケモンには入れ替えできない。 |
| **C-05** 相手のベンチポケモンを 1 匹選び、バトルポケモンと入れ替える | 選んだベンチをバトル場に。ワザの効果を受けないベンチを選んだ場合は入れ替えできない。 |
| **C-06** HP を「●●」回復する | ダメカンをとりのぞく。のっているダメカンが回復量より少ない場合はすべてとりのぞく。ダメカンがのっていないポケモンを選んで使うことは原則できない。 |
| **C-07** ダメカンを●●個のせる | 10 ダメージぶんのダメカンを指定個数のせる。残り HP 以上なら【きぜつ】。ワザの場合は 5. ダメージ以外の効果で行い、弱点・抵抗力・受けている効果は計算しない。ワザの効果を受けないポケモンにはのせられない。 |
| **C-08** ●●にのっているダメカンを▲▲個選び、■■にのせ替える | ダメカンを移動。■■に該当するポケモンがいない場合はできない。のせ替え先がワザの効果を受けない場合ははたらかない。 |
| **C-09** ●●エネルギーを▲▲枚選び、■■につける | 自分のエネルギーは自分のポケモンにしかつけられない。 |
| **C-10** ●●エネルギーを▲▲個選び、■■につけ替える | エネルギーを移動。■■に該当するポケモンがいない場合はできない。つけられないポケモンを選んだ場合、特殊エネルギーはトラッシュされる。 |
| **C-11** ●●から【たね】ポケモンを▲▲枚選び、ベンチに出す | ベンチに空きがないときはワザなら出さず終了、トレーナーズ・特性は使えない。 |
| **C-12** のせて進化させる | 場のポケモンに進化カードを重ねて進化。特別な指示がなければ最初の番・出したばかりでも可。 |
| **C-13** ポケモンの上から「進化カード」を 1 枚はがして退化させる | 上から 1 枚はがす。退化後もダメカンは引きつぐ。場に直接出した進化ポケモンは退化できない。 |
| **C-14** にげられない | そのポケモンは【にげる】を使えない。ベンチにもどる・進化・退化でなくなる。グッズなどでの入れ替えは可。 |
| **C-15** ワザが使えない | エネルギーがついていても宣言して使えない。ベンチにもどる・進化・退化でなくなる。 |
| **C-16** ワザのダメージを受けない | 最終ダメージが 0 となり計算終了。「かかっている効果を計算しない」ワザでは無視されてダメージを受ける。 |
| **C-17** ワザの効果を受けない | 5. ダメージ以外の効果を行わずに手順を進める。すでに受けている効果はなくならない。 |
| **C-18** このワザとして使う | 選んだワザのダメージ・効果のすべてを行う。特別な指示がなければ必要なエネルギーがついていなくても可。 |
| **C-19** 相手（おたがいのプレイヤー）は手札から●●を出して使えない | ポケモンではなく**プレイヤー**が受ける効果。ワザの効果を受けないポケモンがいてもプレイヤーには与えられる。 |
| **C-20** ●●を見る（●●を見て） | ウラのカードのオモテを見る。特別な指示がなければ再びウラにもどす。 |

## D 用語

| 項目 | 内容 |
|------|------|
| **D-01** ワザは失敗 | ワザのダメージ・効果がすべてなくなる。相手を失敗させる効果は 2. で処理。失敗した場合ワザは使ったことにならない。 |
| **D-02** 好きなだけ | 上限なく任意で選べる。ワザで選ぶ場合は 0 でも可。トレーナーズ・特性で 0 で変化がなければ使えない。 |
| **D-03** すべて | 指定されたものを残らず選ぶ。 |
| **D-04** 選び（選ぶ） | 指定数ぶん選択。指定数必ず選ぶ。足りない場合は一番近い数。1 つもない場合は選べない。 |
| **D-05** ●●を▲▲まで | 1 からその数の範囲で任意に選べる。トレーナーズ・特性で選べないと変化がなければ使えない。ワザでは 0 でも可の場合あり。 |
| **D-06** ●●を（が）持つ | カードに書かれているワザ・弱点・抵抗力など。効果で使えるようになったワザは「持つ」には含まれない。 |
| **D-07** ●●タイプ | ポケモンやエネルギーの属性（色）。 |
| **D-08** すべてのタイプのエネルギー | 9 タイプのエネルギーのこと。 |
| **D-09** ワザを使うためのエネルギー | ワザ名の左にあるマークのタイプ。 |
| **D-10** にげるためのエネルギー | カード右下の【にげる】に書かれているエネルギー。 |
| **D-11** 多くなる | 書かれている数字より多いものとして扱う。複数は合計。 |
| **D-12** 少なくなる | 書かれている数字より少ないものとして扱う。複数は合計。 |
| **D-13** なくなる | ないものとして扱う。「なくなる」と「多くなる／少なくなる」が重なった場合は**なくなるが優先**。 |
| **D-14** まんたん | ダメカンが 1 個ものっていない状態。 |
| **D-15** 最大 HP | 場のポケモンの HP に書かれている数字。効果で増減するのは場にいるポケモンのみ。 |
| **D-16** 残り HP | 最大 HP − 受けているダメージ。 |
| **D-17** すでにとったサイドの枚数 | 対戦準備で置いたサイドのうち、とり終えた枚数。60 枚デッキなら 6 から引く。 |
| **D-18** ルールを持つポケモン | カードに「●●（の）ルール」と書かれているポケモン。 |
| **D-19** 引きつぐ | 重ねたり入れ替えたりしたとき、一方が受けていた効果などをもう一方に受けさせる。その番にどの特性を使ったかも引きつぐ。 |

## E その他の説明文

| 項目 | 内容 |
|------|------|
| **E-01** 次の相手の番 | そのワザを使った番の次の、対戦相手の番。ポケモンチェックは含まない。 |
| **E-02** 次の自分の番 | そのワザを使った番の次の、自分の番。 |
| **E-03** ワザのダメージを受けたとき | 10 以上のワザダメージを受けたとき。0 以下になった場合は受けたことにならない。 |
| **E-04** 【きぜつ】したとき | 残り HP 以上で【きぜつ】したとき。【きぜつ】の処理手順の 2. で、トラッシュする直前に行う。 |
| **E-05** 手札から出して進化させたとき | 手札から出して進化させたときにしか行えない。「ふしぎなアメ」で進化した場合も含む。 |
| **E-06** 手札からベンチに出したとき | 手札から【たね】をベンチに出したとき。 |
| **E-07** エネルギーをつけるたび（つけたとき） | 手札からつける・カードの効果でつけたとき。複数枚同時なら枚数ぶんはたらく。 |
| **E-08** ポケモンチェックのたび | ポケモンチェックのときにはたらく。順番は次の番を行うプレイヤーが決められる。 |
| **E-09** ダメージを与える前に | ワザを使う手順の 3. で行う。 |
| **E-10** のぞむなら | 行うか行わないかプレイヤーが選べる。「その場合」に従えない場合はその効果を行わない。 |
| **E-11** 前の相手の番に自分のポケモンが【きぜつ】していたなら | 条件の確認。複数きぜつしていても効果は 1 度だけ。ベンチにもどっても効果は行う。 |
| **E-12** 自分の番に●●回使える | 特性の使用回数。「1 回」は番ごと。同じ名前の特性が複数匹いても 1 匹につき 1 回。場を離れて再び出た場合は別ポケモンとしてまた使える。 |
| **E-13** このポケモンがいるかぎり | そのポケモンが場にいる間にはたらく特性。 |
| **E-14** この特性は別の「●●」を使った番は使えない | 同じ名前の特性を同じ番に 2 回使えない。使ったポケモンが場を離れても、同じ番で別のポケモンは使えない。 |
| **E-15** この特性の効果は重ならない | 同じ特性のポケモンが何匹いても 1 匹ぶんの効果のみ。 |
| **E-16** ●●でのせるダメカンの数は（が）▲▲個になる | 特殊状態のダメカン数を変更。複数はたらく場合は**最も大きい数値**が 1 回だけ適用。 |
| **E-17** ●●でのせるダメカンの数は（が）▲▲個多くなる | 特殊状態のダメカン数を追加。複数は合計。「なる」と「多くなる」が重なった場合は先に「なる」を適用し、その後「多くなる」を合計。 |
| **E-18** このカードが●●にあるなら、自分の番に 1 回使える | 場以外の場所（手札など）にあるときに使える特性。 |
| **E-19** そして山札を切る | 山札を見たあともどすときに切る。内容を見た場合は指定のカードを選ばなくても原則切る。 |
| **E-20** ●●する。その後、▲▲する | 前文を 1 つでも行えたら「その後」も行う。全く行えなかったら「その後」は行わない。前文だけ行うことは可（「その後」に従えなくても）。 |
| **E-21** 番の終わり | ワザの処理後、またはワザを使わずに番を終えるタイミング。ポケモンチェックの前。複数効果はその番を行うプレイヤーが順番を決める。 |
| **E-22** 次の自分の番の終わりまで | その自分の番から、次の自分の番の終わりまで。ポケモンチェックの間もはたらく。 |
| **E-23** 次の相手の番の終わりまで | その自分の番から、次の相手の番の終わりまで。 |
| **E-24** この番に、このポケモンが●●から進化していたなら | ワザを使う番に、そのポケモンが進化しているか。ベンチにもどっていても効果は行う。 |
| **E-25** この番に、このポケモンがベンチからバトル場に出ていたなら | ベンチからバトル場に出た手段は問わない。何度出ていても効果は 1 度だけ。 |
| **E-26** この番に、手札からサポートを出して使っていたなら | 手札からサポートを使ったかどうか。 |
| **E-27** この番に、このポケモンの HP を回復していたなら | そのポケモンの HP を 10 以上回復したか。複数回・ベンチ移動後でも 1 度だけ。 |
| **E-28** このワザは、●●でも使える | 通常ワザが使えない状況でも、指定の状況を満たせば使える。 |
| **E-29** ●●が手札から▲▲を出して使ったとき、その効果を受けない | そのカードの効果は受けないが、カード自体は使える。その場合はそれ以外の内容を行う。 |
| **E-30** 自分の番に、自分の手札を●●枚トラッシュするなら、1 回使える | トラッシュすることが条件。従えない場合は使えない。 |
| **E-31** 手札に加える前に | 山札を引く・サイドをとったとき、オモテを見たあと手札に加える前のタイミング。 |
| **E-32** このワザは、●●なら、▲▲エネルギー■■個で使える | 条件を満たすと必要なエネルギーが少なくなる。「多くなる」効果の影響を受けない。 |
| **E-33** 自分の手札から●●を▲▲枚（まで）選ぶ | 手札から指定を「まで」の場合は少ない枚数・0 枚可。「好きな」の場合は指定枚数必ず。 |
| **E-34** ウラになっている自分のサイドから●●を▲▲枚選ぶ | サイドから選ぶ。枚数は E-33 と同様。 |
| **E-35** ●●をウラにして切り（切る） | ウラにしてよく混ぜる。オモテを見てはいけない。●●が 1 枚もない場合は切れない。 |
| **E-36** ●●を山札にもどして切る | もどして山札を切る。合計が 1 枚以上あれば切ったことになる。「▲▲枚選ぶ」で選べるものが 0 の場合は切らない。 |
| **E-37** ●●にいる（ある）ワザ「▲▲」を持つポケモン | その場所にいる・ある、そのワザ名を持つポケモン。効果で使えるようになったワザは「持つ」に含まれない。 |
| **E-38** この対戦は●●の勝ち（負け）になる | 通常の勝敗判定以外の方法で決着。プレイヤーにかかる効果ではない。 |
| **E-39** 「基本●●エネルギー」を▲▲ | カード名が「基本●●エネルギー」であるカードに▲▲の処理。タイプが効果で変わっていてもカード名が同じなら受ける。 |
